# 🧠 Data Generation Philosophy

The dataset is generated using a structured, clinically coherent sequence rather than independent random sampling.

Generation order:

1. Demographics & Social Determinants  
2. Comorbidities  
3. Acute physiology  
4. Stroke severity (NIHSS)  
5. Treatment allocation  
6. Early rehabilitation processes  
7. Complications  
8. Depression & nutrition  
9. Functional outcomes (ADL discharge → ADL 6 months)

This preserves directional clinical plausibility:
- Higher NIHSS increases complication risk  
- Complications worsen nutrition and depression  
- Early rehabilitation improves outcomes  
- Discharge ADL anchors 6-month ADL

# 📊 Variables Generated

## 1️⃣ Demographics / SDOH

- `Sex` ("1" = male, "0" = female)  
- `Age` (18–100 integer)  
- `Age_cat` ("0" <75, "1" ≥75)  
- `Socioeconomic` (quintile "1"–"5")  
- `Education` ("0" primary → "3" university)  
- `Martial status` ("1" married, "0" single)  
- `Caregiver` ("1" yes, "0" no)  
- `Smoking` ("1" yes, "0" no)  
- `HealthInsurance`  
  - "1" Universal Coverage  
  - "2" Social Security  
  - "3" CSMBS / Direct billing  
  - "4" Other  

Insurance influences treatment probability through an access proxy.

---

## 2️⃣ Comorbidities

Binary variables:

- `HT`  
- `DM`  
- `DLP`  
- `AF`  

Generated via logistic functions depending on age, sex, SES.

---

## 3️⃣ Acute Physiology

- `SBP`, `DBP` (continuous)  
- `SBP_cat`, `DBP_cat` (thresholded)  
- `HbA1c`  
- `Glucose`  
- `SHR` (Stress Hyperglycemia Ratio) tertiles  

SHR encoding:

- "0" = T1  
- "1" = T2  
- "2" = T3  

---

## 4️⃣ Stroke Severity (NIHSS)

Generated using:

- Negative binomial baseline  
- Age + AF + comorbidity shift  
- Large vessel occlusion bump  
- Random noise  

Categorization:

- `NIHSS_cat = "0"` (<15)  
- `NIHSS_cat = "1"` (≥15)  

---

## 5️⃣ Treatment Allocation

Encoded as:

- "0" Conservative  
- "1" rTPA  
- "2" Thrombectomy  

Rules:

- NIHSS <4 → no rTPA  
- BP eligibility threshold  
- Thrombectomy probability increases with severity  
- Mutually exclusive assignment  

---

## 6️⃣ Early Processes

- `EarlyRehabilitation`  
- `EarlyMobilization`  

Probability decreases with severity and increases with caregiver support.

---

## 7️⃣ Complications

Generated via logistic functions influenced by severity and age:

- Pneumonia  
- UTI  
- Sepsis  
- VTE  
- Delirium  
- HF  
- Large infarction  
- Hemorrhagic transformation  
- Brain herniation  
- Neurosurgery requirement  

---

## 8️⃣ Depression (PHQ-9)

Generated via Poisson model with severity influence.

Categories:

- "0" Minimal  
- "1" Mild  
- "2" Moderate  
- "3" Severe  

Clinical depression flag defined as PHQ9 ≥ 7.

---

## 9️⃣ Nutrition (MNA-like 0–30)

Score influenced by:

- Severity  
- Sepsis  
- Pneumonia  
- Rehabilitation  

Categories:

- "0" Normal (24–30)  
- "1" At risk (17–23)  
- "2" Malnourished (≤16)  

6-month nutrition score also generated.

---

# 🏥 Functional Outcomes

## ADL at Discharge

Continuous score: 0–20  

Rebalanced across dependency bands:

- 0–4  
- 5–8  
- 9–11  
- ≥12  

Dependency cut-off:

```
ADL_dependent_cut12 = 1 if ADL ≤ 11
```

---

## ADL at 6 Months (Primary Outcome Proxy)

Linear score model anchored to discharge ADL:

```
ADL_6m =
  beta_anchor_dis * ADL_Discharge
+ beta_anchor_const
+ demographic effects
+ comorbidity effects
+ NIHSS effects
+ SHR effects
+ complication effects
+ depression effects
+ nutrition effects
+ treatment effects
+ rehabilitation effects
+ noise
```

Clipped to 0–20 and rounded to integer.

Binary outcome:

```
ADL_6month_dependent_cut12 = 1 if ADL_6month ≤ 11
```

# 📎 METHODS APPENDIX

## A. Study Design Emulation

The synthetic dataset emulates a **prospective cohort study** of acute ischemic stroke patients with:

- Baseline NIHSS severity  
- In-hospital events  
- Rehabilitation processes  
- 6-month functional outcome  

Primary endpoint proxy:

- Functional dependency at 6 months  
- Defined as ADL <12  

Subgroup structure preserved:

- NIHSS <15 vs NIHSS ≥15  

---

## B. Statistical Generation Framework

### 1. Logistic Variable Generation

Binary variables are generated using:

```
P(Y=1) = sigmoid(linear predictor)
```

Where:

```
sigmoid(x) = 1 / (1 + exp(-x))
```

---

### 2. Severity Modeling

NIHSS generation:

- Negative binomial base distribution  
- Additive severity shift  
- Large-vessel occlusion bump  
- Random variation  

Produces right-skew severity distribution.

---

### 3. Treatment Selection Model

Treatment probabilities depend on:

- NIHSS  
- BP eligibility thresholds  
- Insurance access proxy  

This preserves realistic selection bias structure.

---

### 4. Outcome Model

ADL at 6 months follows:

```
Y = α + β1X1 + β2X2 + ... + ε
```

Where:

- α = anchoring intercept  
- β = tunable coefficients  
- ε ~ Normal(0, noise_sd²)  

---

## C. Calibration Logic

Discharge ADL distribution is rebalanced by:

1. Ranking individuals by severity index  
2. Assigning ADL categories proportionally  
3. Sampling uniformly within clinical bands  

Ensures realistic dependency proportions.

---

## D. Reproducibility

All simulations use:

```

```

Identical seeds produce identical datasets.
